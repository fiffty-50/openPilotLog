void JsonHelper::exportDatabase ( )
static

exportDatabase exports the currently active database to JSON.

QJsonObject can be constructed from QHash<QString, QVariant> aka QVariantmap. All rows are put into a QJsonArray and the Array is embedded into a QJsonDocument, which can be exported. The files are created at Paths::Export.
